## Setup

### Dev

```sh
# (Linux) Export UID to fix permissions
export UID

# Boot the stack; this will
# - provision mongo
# - launch api & web
npm run up

# Only run api & web
npm run dev

# Stop containers
npm run stop

# Tear down containers
npm run down
```

## Stack

### Backend

- Node + Express
- GraphQL + Apollo Server Express
- MongoDB + Mongoose

### Frontend

- React 17.0.2
- Apollo Client 3.4.16
- Material-UI 4.11.4
